Before we dive into the blocks themselves, we must talk about the types of statistical variables we can visualize.

Table columns contain data that can be considered either quantitative or qualitative variables. We use the word variable here to indicate that you can treat the entire column like you would treat any one if its values when writing code.

  • Quantitative variables are numerical variables we can do arithmetic with.
  • Qualitative variables are categorical variables we can not do arithmetic with

This distinction between the two types of variables is important from a programming perspective when choosing which kind of visualization technique we should use to represent a given query's output.

Each variable type can be further split into 2 subgroups:

  • Discrete quantitative variables are numerical variables whose values come in discrete units (i.e. the number of passengers in a taxi ride, number of students in a class, etc.)
  • Continuous quantitative variables are numerical variables whose values lie on some point of a continuous spectrum (i.e. cost of a taxi ride, GPA, etc.)
  • Ordinal qualitative variables are categorical variables whose values can be ordered in some non-numerical way (i.e. level of education (elementary, middle, high, ...) , etc.)
  • Nominal qualitative variables are categorical variables whose values cannot be ordered (i.e. color of a taxi, political affiliation, etc.)

For each data visualization block we cover in this assignment, we will also talk about which types of variables are associated with that visualization technique. It is always important to remember which types of data should be represented in certain ways.
