In this assignment we will be exploring data visualization. Data visualization is important for being able to display data analysis in a way that anyone, including people who do not have a data science background, can understand and learn from. It is also important as a tool that helps data scientists learn about their data and draw data-driven conclusions.

Data can be visualized in many different ways depending on the content of the data and the intended conclusion. As we answer questions about data using the querying blocks we have learned about in previous assignments, we can also choose how we want to visually represent those answers to other data scientists or to the general public. This means that we not only need to understand how the visualization blocks themselves work, but also how to best use them in practice.

In this assignment, we will be introducing 4 data visualization techniques in DASIS: horizontal bar, histogram, scatter plot, and line plot. Each of these visualizations has different uses that we will discuss in this assignment.
